# WhatsBiz CRM — Demo Script (Dicoding Presentation Flow)

---

# OBJECTIVE DEMO

Tujuan demo:

* Menunjukkan masalah nyata UMKM
* Menampilkan solusi end-to-end
* Memperlihatkan AI + automation value
* Membuat juri “merasakan” produk, bukan sekadar melihat UI

---

# TOTAL DEMO TIME

8 – 12 menit ideal

---

# SCENE STRUCTURE

Demo dibagi menjadi 6 skenario utama:

1. Problem Simulation
2. Customer Chat Entry (WhatsApp Inbox)
3. AI Response + Context Panel
4. Order Creation Flow
5. Invoice & Payment Tracking
6. Automation Trigger + Insight

---

## Narasi (Presenter)

> "Bayangkan Anda punya bisnis laundry kecil. Setiap hari masuk puluhan chat WhatsApp, tapi semua bercampur dengan chat pribadi."

> "Tidak ada sistem. Tidak ada pencatatan. Tidak ada follow-up."

## Visual

* Tampilkan WhatsApp chat berantakan (simulasi)

# TIPS PRESENTASI

## DO:

* Fokus ke storytelling
* Tunjukkan alur end-to-end
* Tekankan “before vs after”
* Gunakan demo real-time

---

## DON'T:

* Jangan jelaskan terlalu teknis
* Jangan fokus ke code
* Jangan terlalu lama di satu fitur

---

# SUCCESS CRITERIA

Jika demo berhasil, juri harus merasa:

* "Ini berguna banget untuk UMKM"
* "Ini sudah siap dipakai"
* "Ini beda dari CRUD biasa"
